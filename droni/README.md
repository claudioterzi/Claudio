## Regolamento EASA in 3 righe

- **Open** (< 900g, < 120m, distanza da persone): nessuna autorizzazione, solo registrazione
- **Specific** (operazioni rischiose): autorizzazione ENAC/autorità nazionale richiesta
- **Certified**: equivalente aviazione civile — per droni cargo, passeggeri, sistemi autonomi critici
