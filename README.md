# 🎮 LGAI - Life Game AI Sistema Completo

## 📦 Installazione Rapida

### Requisiti
- Python 3.7+ (nessuna dipendenza esterna!)

### Setup (2 passi)
```bash
# 1. Estrai l'archivio
tar -xzf LGAI-Complete-System.tar.gz

# 2. Inizia!
cd cli
python lgai.py status
python lgai.py checkin 8 7 --note "Primo giorno!"
```

## 🚀 Quick Start

### Workflow Giornaliero
```bash
# Mattina
python lgai.py checkin [mood 1-10] [energia 1-10] -i

# Sera  
python lgai.py checkout --habits-positive "1,6,13" --habits-negative "101"
```

### Comandi Principali
- `status` - Stato completo
- `habits` - 46 abitudini tracciabili
- `missioni` - 45 missioni
- `shop` - 50 ricompense Baros
- `predict` - Predizioni quantiche
- `talk "domanda"` - Parla con Raffaello
